## Getting Started

### Clone The Repository

```bash
git clone 
```

### Go to the Directory

```bash
cd mesh-web3-login-system
```

### Install Dependencies:

```bash
yarn
```

_or_

```bash
npm i
```

### Run The Development Server

```bash
yarn dev
```

_or_

```bash
npm run dev
```
